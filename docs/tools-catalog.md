# ツールカタログ

Generator Tools で提供する全ツールの一覧。
既存ツールと追加候補をカテゴリ別に整理する。

参考: [DevToys](https://devtoys.app/), [IT-Tools](https://it-tools.tech), [CyberChef](https://www.cyberchef.dev/), [transform.tools](https://transform.tools), [JSON Knife](https://tools.shanecode.org/)

---

## 凡例

- **実装済** = 既にアプリに存在するツール
- **追加候補** = 未実装の追加候補ツール
